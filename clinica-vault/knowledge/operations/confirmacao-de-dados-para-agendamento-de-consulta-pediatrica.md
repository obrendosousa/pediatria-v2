## Pergunta

Confirmação de dados para agendamento de consulta pediátrica.

## Resposta

Para confirmar o agendamento, preciso apenas dos dados do bebê: Nome completo e Data de Nascimento. O pagamento pode ser feito na recepção ou via Pix agora para adiantar.
